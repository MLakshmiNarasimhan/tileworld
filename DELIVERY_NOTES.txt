Tileworld strict range-3 survival package

Contents
- src/ : current validated source
- bin/ : compiled classes from build_opt9
- mason.jar : simulator dependency

Run
- javac -encoding utf8 -cp mason.jar -d bin @all_sources.txt
- use a Java 25 runtime for the packaged bin in this workspace
- java -cp "bin;mason.jar" tileworld.TileworldMain

Validation summary
- 50x50: average reward 426.9 over 10 random seeds, no fuel-runout messages observed
- 80x80: average reward 241.1 over 10 random seeds, no fuel-runout messages observed
- 150x150: average reward 34.9 over 10 random seeds, no fuel-runout messages observed

Important note
- This package keeps strict legal sensing range 3 and one action per tick.
- 150x150 now survives the tested 10-seed sample without fuel-runout messages, but its reward remains much lower than the smaller maps under the unchanged simulator rules.
